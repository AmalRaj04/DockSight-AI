# Tech Stack

## Backend

- Python 3.11
- FastAPI (API + orchestration)
- RDKit (cheminformatics)
- Biopython (protein parsing)
- NumPy / Pandas (numerical analysis)

## AI / Reasoning

- LLM via API (OpenAI / Claude / Groq)
- Prompt-driven scientific reasoning
- Rule-based validation + heuristics

## Agent Orchestration

- Custom pipeline-based agent
- Deterministic state machine (no auto creativity)
- Optional: LangGraph (not mandatory)

## Visualization

- PyMOL (static images)
- Mol\* or NGL Viewer (web-based 3D)
- Matplotlib / Plotly (charts & tables)

## Frontend

- React + Vite
- TailwindCSS
- Mol\* Viewer integration
- PDF / DOCX preview

## Blockchain

- Solana
- Anchor framework
- On-chain metadata storage (hashes, timestamps)

## Dev Tooling

- Kiro IDE
- ESLint / Prettier
- Git
